# LDJAM 2025 Project

Unity 6.2 game project for Ludum Dare Jam.

## Setup
- Unity Version: 6.2
- Clone repo
- Open with Unity Hub

## Rules
- ❌ No game code before jam start
- ✅ Allowed: repo setup, docs, planning, wireframes
- Feature branches: `feature/<system>-<name>`
- Main branch protected, `dev` for integration
